# Machine-Learning-vs-Deep-Learning
Deep Learning (DL) is actually a subset of Machine Learning (ML). Artificial Intelligence (AI)>Machine Learning (ML)>Deep Learning (DL). Both are used to make systems learn patterns from data, but they differ in:  Architecture, Data requirements, Complexity Performance and Hardware needs
